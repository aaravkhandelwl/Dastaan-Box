 # Dastaan Box 🪆

An offline AI-powered Kathputli folk puppet that brings 
grandparents' stories to life for children — no internet required.

## What it does
- Child presses a button → puppet narrates a folk story
- Captures grandparent voice recordings via INMP441 mic
- Plays back stories offline using ESP32 + SD card
- Supports regional Indian languages via Bhashini API

## Hardware
- ESP32 DevKit V1 (brain)
- INMP441 I2S Microphone (voice capture)
- PAM8403 Amplifier + 3W Speaker (audio output)
- MicroSD Module (offline story storage)
- Li-Ion 18650 Battery (fully wireless)
